Number Puzzle

How to Play

  • Slide tiles into order — 1, 2, 3… left-to-right, top-to-bottom, with the empty space at the bottom-right.
  • Tap any tile that shares a row or column with the empty space to slide it.
  • All tiles between the tapped tile and the empty space shift over in one smooth move.
  • Use arrow keys on a keyboard, or swipe on touch screens.
  • Fewer moves and less time is better!

How it Works

The computer stores all the tiles in a list (like a row of numbered boxes). The empty space is just a box holding the number zero.

  • When you tap a tile, the computer checks: is it in the same row or column as the zero box?
  • If yes, it swaps the numbers along that row or column — like sliding books along a shelf — until the zero reaches the tile you tapped.
  • Then it redraws the board by updating each tile's position using CSS — the styling language that controls how things look on a web page.
  • A tiny timer runs every half-second to count how long you've been playing!

Everything you see — the colours, the animations, the sounds — is written in just three languages: HTML (structure), CSS (appearance), and JavaScript (behaviour).

The board is a flat array of N² integers (0 = blank). All game state lives in one object.

  • Move validation: a tap is legal only when the tile's row or column matches the blank's — checked in O(1).
  • Shift logic: tiles between the tap and blank are shifted by one index in a linear scan — no swaps, no splice, just a while-loop.
  • Rendering: tiles are absolutely-positioned via CSS custom properties (--row, --col). Updating those properties triggers a CSS transition automatically — no JS animation loop needed.
  • Solvability: at startup the shuffle is validated: odd grids need an even inversion count; even grids need (inversions + blank's row from bottom) to be odd.
  • Sound: a synthesised tock is built with the Web Audio API — white noise through a bandpass filter with a 2 ms attack / 55 ms exponential decay, scheduled to fire exactly when the CSS transition ends (120 ms).
