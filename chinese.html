Where do I get these?
  1. Deploy modal/azure_speech_proxy.py from the health repo to Modal (it auto-deploys on push, or run modal deploy modal/azure_speech_proxy.py)
  2. Create the azure-speech-proxy-secrets Modal secret with an Azure Speech key, region, and a proxy token you choose (see the file's docstring)
  3. Paste the URL Modal prints into the box above, and the proxy token into API Key
Only the proxy URL and proxy token are stored in this browser (localStorage) — the real Azure key stays server-side in the Modal secret.
